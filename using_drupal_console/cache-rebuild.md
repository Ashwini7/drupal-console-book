# cache:rebuild
The **cache:rebuild** command rebuilds and clears all site caches

```
$ drupal cache:rebuild --help
```
**Usage:**
```
$ drupal cache:rebuild [--cache[="..."]]
```

**Aliases:**
```
$ drupal c:r --help
```

## Available options
Options | Details
------------ |-------------
--cache       |        Only clear a specific cache. (default: "")
