# test:debug
The **test:debug** command lists unit tests available to the application

```
$ drupal test:debug --help
```
**Usage:**
```
$ drupal test:debug [--group[="..."]] [test-id]
```
## Arguments
Argument | Details
------------ |-------------
test-id      |         commands.test.debug.arguments.resource-id

## Available options
Options | Details
------------ |-------------
--group      |         Group
