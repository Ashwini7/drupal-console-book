# rest:debug
The **rest:debug** command displays current migration available for the application

```
$ drupal rest:debug --help
```
**Usage:**
```
$ drupal rest:debug [--authorization[="..."]] [resource-id]
```
## Arguments
Argument | Details
------------ |-------------
 resource-id  |         Rest ID

## Available options
Options | Details
------------ |-------------
 --authorization |      REST resource status enabled / disabled
