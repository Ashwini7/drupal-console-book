# rest:debug
The **rest:debug** command displays current migration available for the application

```
$ drupal rest:debug --help
```
**Usage:**
```
$ drupal rest:debug [--autorization[="..."]] [resource-id]
```
## Arguments
Argument | Details
------------ |-------------
 resource-id  |         Rest ID

## Available options
Options | Details
------------ |-------------
 --autorization |      REST resource status enabled / disabled
