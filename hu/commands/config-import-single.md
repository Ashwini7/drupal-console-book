# config:import:single
The **config:import:single** command Importálja a kijelölt konfigurációt.

**Usage:**
```
$ drupal config:import:single [arguments] 
$ cis  
```

## Available arguments
Argument | Details
---------|-------------
config-name | Konfiguráció neve.
input-file | Import fájlok felé vezető útvonal.
