# generate:doc:dash
**generate:doc:dash** parancs DrupalConsole.docset csomag létrehozása a Dash számára

**Használat:**
```
$ drupal generate:doc:dash [options] 
$ gdd  
```

## Rendelkezésre álló beállítások
Beállítás | Részletek
-------|-------------
--path | A docset mentési könyvtárának útvonala.
