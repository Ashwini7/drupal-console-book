# user:debug
The **user:debug** command Megjeleníti az alkalmazás aktuális felhasználóit

**Usage:**
```
$ drupal user:debug [options] 
```

## Available options
Option | Details
-------|-------------
--roles | Szerepek szűrése a hibakereséshez
--limit | Hány felhasználó jelenjen meg a hibakeresésben
