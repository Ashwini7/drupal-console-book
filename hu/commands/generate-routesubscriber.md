# generate:routesubscriber
**generate:routesubscriber** parancs RouteSubscriber létrehozása

**Használat:**
```
$ drupal generate:routesubscriber [options] 
```

## Rendelkezésre álló beállítások
Beállítás | Részletek
-------|-------------
--module | A modul neve.
--name | Szolgáltatásnév
--class | Osztálynév
