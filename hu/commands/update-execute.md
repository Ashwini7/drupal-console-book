# update:execute
The **update:execute** command Egy modul adott Update N függvényének végrehajtása, vagy az összes végrehajtása

**Usage:**
```
$ drupal update:execute [arguments] 
$ upe  
```

## Available arguments
Argument | Details
---------|-------------
module | A Modul neve.
update-n | Végrehajtandó adott Update N függvény
