# generate:plugin:imageeffect
The **generate:plugin:imageeffect** command Kép hatás beépülő létrehozása.

**Usage:**
```
$ drupal generate:plugin:imageeffect [options] 
```

## Available options
Option | Details
-------|-------------
--module | A Modul neve.
--class | Plugin class neve
--label | Plugin címke
--plugin-id | Plugin ID
--description | Plugin leírás
