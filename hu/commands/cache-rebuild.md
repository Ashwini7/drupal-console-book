# cache:rebuild
The **cache:rebuild** command Minden gyorsítótár újraépítése és törlése.

**Usage:**
```
$ drupal cache:rebuild [arguments] 
$ cr  
```

## Available arguments
Argument | Details
---------|-------------
cache | Csak bizonyos gyorsítótár törlése.

## Examples
* Rebuild all caches
```
$ drupal cr all
```
* Rebuild discovery cache
```
$ drupal cr discovery
```
