# user:delete
The **user:delete** command Felhasználók törlése az alkalmazásból

**Usage:**
```
$ drupal user:delete [options] 
```

## Available options
Option | Details
-------|-------------
--user-id | Törölni kívánt felhasználói azonosító
--roles | A törölni kívánt felhasználókkal társított szerepkörök
