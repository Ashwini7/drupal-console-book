# generate:permissions
**generate:permissions** parancs Modul engedélyeinek létrehozása

**Használat:**
```
$ drupal generate:permissions [options] 
$ gp  
```

## Rendelkezésre álló beállítások
Beállítás | Részletek
-------|-------------
--module | A modul neve.
--permissions | Engedélyek létrehozása.
