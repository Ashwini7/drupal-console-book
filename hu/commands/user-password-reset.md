# user:password:reset
The **user:password:reset** command Addott felhasználó jelszavának alaphelyzetbe állítása.

**Usage:**
```
$ drupal user:password:reset [arguments] 
$ uspr  
```

## Available arguments
Argument | Details
---------|-------------
user | Felhasználói azonosító
password | Szövegformátumú jelszó
