# cache:rebuild
The **cache:rebuild** command सभी साइट caches को पुनर्निर्माण और साफ़ करें।

**Usage:**
```
$ drupal cache:rebuild [arguments] 
$ cr  
```

## Available arguments
Argument | Details
---------|-------------
cache | केवल एक विशेष cache साफ़ करें।

## Examples
* Rebuild all caches
```
$ drupal cr all
```
* Rebuild discovery cache
```
$ drupal cr discovery
```
