# module:download
The **module:download** command मोड्यूल या मोड्यूलो को डाउनलोड करें

**Usage:**
```
$ drupal module:download [arguments] 
$ md  
```

## Available arguments
Argument | Details
---------|-------------
module | commands.module.download.options.module
version | मोड्यूल संस्करण अथवा 1.x-dev
