# database:dump
The **database:dump** command Dump structure and contents of MySQL databases and tables

**प्रयोग:**
```
$ drupal database:dump [arguments] [options] 
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--file | commands.database.dump.option.file

## उपलब्ध तर्कों  
तर्क | विवरण
---------|-------------
database | Database key from settings.php
