# test:debug
The **test:debug** command अनुप्रयोग की सारी उपलब्ध परिक्षण यूनिटो को दिखाएँ।

**प्रयोग:**
```
$ drupal test:debug [arguments] [options] 
$ td  
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--group | समूह

## उपलब्ध तर्कों  
तर्क | विवरण
---------|-------------
test-class | परिक्षण क्लास
