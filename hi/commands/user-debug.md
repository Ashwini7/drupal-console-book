# user:debug
The **user:debug** command Displays current users for the application

**प्रयोग:**
```
$ drupal user:debug [arguments] [options] 
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--limit | How many users would you listed in debug

## उपलब्ध तर्कों  
तर्क | विवरण
---------|-------------
roles | Roles to filter debug
