# theme:install
The **theme:install** command Install theme or themes in the application

**प्रयोग:**
```
$ drupal theme:install [arguments] [options] 
$ ti  
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--set-default | Set theme as default theme

## उपलब्ध तर्कों  
तर्क | विवरण
---------|-------------
theme | commands.theme.install.options.module
