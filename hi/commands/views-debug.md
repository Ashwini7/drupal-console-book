# views:debug
The **views:debug** command अनुप्रयोग के वर्तमान व्यूज साधनो को दिखाएँ

**प्रयोग:**
```
$ drupal views:debug [arguments] [options] 
$ vde  
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--tag | View tag
--status | View status (Enabled|Disabled)

## उपलब्ध तर्कों  
तर्क | विवरण
---------|-------------
view-id | View ID
