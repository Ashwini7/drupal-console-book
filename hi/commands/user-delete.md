# user:delete
The **user:delete** command Delete users for the application

**प्रयोग:**
```
$ drupal user:delete [arguments] [options] 
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--roles | commands.user.debug.options.roles

## उपलब्ध तर्कों  
तर्क | विवरण
---------|-------------
user-id | User id to be deleted
