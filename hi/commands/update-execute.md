# update:execute
The **update:execute** command मोड्यूल के किसी विशेष नवीनीकरण N फंक्शन को चलायें या सभी को चलायें।

**प्रयोग:**
```
$ drupal update:execute [arguments] 
$ upe  
```

## उपलब्ध तर्कों  
तर्क | विवरण
---------|-------------
module | मोड्यूल का नाम।
update-n | Specific Update N function to be executed
