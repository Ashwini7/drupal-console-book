# config:diff
**config:diff** comanda Ouput configuration items that are different in active configuration compared with a directory.

**Folosire:**
```
$ drupal config:diff [arguments] [options] 
```

## Opțiuni disponibile
Opțiune | Detalii
-------|-------------
--reverse | See the changes in reverse (i.e diff a directory to the active configuration).

## Argumente disponibile
Argument | Detalii
---------|-------------
directory | The directory to diff against. If ommitted, choose from Drupal config directories.
