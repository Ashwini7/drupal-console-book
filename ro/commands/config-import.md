# config:import
**config:import** comanda Importă configurarea pentru aplicația curentă.

**Folosire:**
```
$ drupal config:import [options] 
$ ci  
```

## Opțiuni disponibile
Opțiune | Detalii
-------|-------------
--file | commands.config.import.option.file
--remove-files | commands.config.import.option.keep-files
