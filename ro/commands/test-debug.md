# test:debug
**test:debug** comanda Afişează testele accesibile

**Folosire:**
```
$ drupal test:debug [arguments] [options] 
$ td  
```

## Opțiuni disponibile
Opțiune | Detalii
-------|-------------
--test-class | Clasa de test

## Argumente disponibile
Argument | Detalii
---------|-------------
group | Grupa
