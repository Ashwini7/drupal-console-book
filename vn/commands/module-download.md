# module:download
Câu lệnh **module:download** Tải module hoặc các modules trong ứng dụng

**Usage:**
```
$ drupal module:download [arguments] 
$ md  
```

## Các đối số có sẵn
Đối số | Các chi tiết
---------|-------------
module | commands.module.download.options.module
version | Phiên bản Module i.e 1.x-dev
