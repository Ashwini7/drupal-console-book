# list
The **list** command Список команд

**Использование:**
```
$ drupal list [arguments] [options] 
```

## Доступные опции
Опция | Описание
-------|-------------
--xml | Вывести список в XML формате
--raw | Вывести список в исходном формате
--format | Формат вывода (txt, xml, json, или md)

## Доступные параметры
Параметр | Описание
---------|-------------
namespace | Пространство имен
