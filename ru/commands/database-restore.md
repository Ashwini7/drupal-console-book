# database:restore
The **database:restore** command Восстановление структуры и контента MySQL БД и таблиц

**Использование:**
```
$ drupal database:restore [arguments] [options] 
```

## Доступные опции
Опция | Описание
-------|-------------
--file | Имя файла бекапа вашей БД

## Доступные параметры
Параметр | Описание
---------|-------------
database | Ключ БД из settings.php
