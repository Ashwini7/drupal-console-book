# views:debug
The **views:debug** command Отображение текущих ресурсов представлений приложения

**Использование:**
```
$ drupal views:debug [arguments] [options] 
$ vde  
```

## Доступные опции
Опция | Описание
-------|-------------
--tag | Тег представления
--status | Статус представления (Вкл.|Выкл.)

## Доступные параметры
Параметр | Описание
---------|-------------
view-id | ID Представления
