# database:dump
The **database:dump** command Дамп структуры и контента MySQL БД и таблиц

**Использование:**
```
$ drupal database:dump [arguments] [options] 
```

## Доступные опции
Опция | Описание
-------|-------------
--file | commands.database.dump.option.file

## Доступные параметры
Параметр | Описание
---------|-------------
database | Ключ БД из settings.php
