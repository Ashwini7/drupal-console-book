# generate:plugin:imageformatter
The **generate:plugin:imageformatter** command Generate image formatter plugin.

**Использование:**
```
$ drupal generate:plugin:imageformatter [options] 
```

## Доступные опции
Опция | Описание
-------|-------------
--module | Имя модуля.
--class | Plugin class name
--label | Plugin label
--plugin-id | Plugin id
