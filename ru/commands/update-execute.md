# update:execute
The **update:execute** command Выполнить конкретный update_N хук в модуле или выполнить все

**Использование:**
```
$ drupal update:execute [arguments] 
$ upe  
```

## Доступные параметры
Параметр | Описание
---------|-------------
module | Имя модуля.
update-n | Конкретный update_N хук будет выполнен
