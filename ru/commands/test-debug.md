# test:debug
The **test:debug** command List Test Units available for the application.

**Использование:**
```
$ drupal test:debug [arguments] [options] 
$ td  
```

## Доступные опции
Опция | Описание
-------|-------------
--test-class | Test Class

## Доступные параметры
Параметр | Описание
---------|-------------
group | Group
