# config:import:single
The **config:import:single** command Импорт выбранной конфигурации.

**Использование:**
```
$ drupal config:import:single [arguments] 
$ cis  
```

## Доступные параметры
Параметр | Описание
---------|-------------
config-name | Имя конфигурации.
input-file | Путь к импортируемым файлам.
