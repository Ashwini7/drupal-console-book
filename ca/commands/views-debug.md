# views:debug
El comandament **views:debug** executa Display current views resources for the application

**Ús:**
```
$ drupal views:debug [arguments] [options] 
$ vde  
```

## Opcions disponibles
Opció | Detalls
-------|-------------
--tag | View tag
--status | View status (Enabled|Disabled)

## Arguments disponibles
Argument | Detalls
---------|-------------
view-id | View ID
