# rest:debug
El comandament **rest:debug** executa Mostra el recurs Rest actual per l'aplicació

**Ús:**
```
$ drupal rest:debug [arguments] [options] 
$ rede  
```

## Opcions disponibles
Opció | Detalls
-------|-------------
--authorization | Rest resource status habilitat | deshabilitat

## Arguments disponibles
Argument | Detalls
---------|-------------
resource-id | Rest ID
