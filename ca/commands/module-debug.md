# module:debug
El comandament **module:debug** executa Mostrar els mòduls disponibles per l'aplicació

**Ús:**
```
$ drupal module:debug [options] 
```

## Opcions disponibles
Opció | Detalls
-------|-------------
--status | Estat del mòdul [habilitat|deshabilitat]
--type | Tipus de mòdul [core|no-core]
