# test:debug
El comandament **test:debug** executa Llista els Test Units disponibles per l'aplicació.

**Ús:**
```
$ drupal test:debug [arguments] [options] 
$ td  
```

## Opcions disponibles
Opció | Detalls
-------|-------------
--test-class | Classe test

## Arguments disponibles
Argument | Detalls
---------|-------------
group | Grup
