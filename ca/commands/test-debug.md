# test:debug
El comandament **test:debug** executa List Test Units available for the application.

**Ús:**
```
$ drupal test:debug [arguments] [options] 
$ td  
```

## Opcions disponibles
Opció | Detalls
-------|-------------
--group | Group

## Arguments disponibles
Argument | Detalls
---------|-------------
test-class | Test Class
