# test:debug
O **test:debug** comando Listar testes unitários disponíveis para a aplicação.

**Uso:**
```
$ drupal test:debug [arguments] [options] 
$ td  
```

## Opções disponíveis
Opção | Detalhes
-------|-------------
--group | Grupo

## Argumentos disponíveis
Argumento | Detalhes
---------|-------------
test-class | Classe do teste
