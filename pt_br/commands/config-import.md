# config:import
O **config:import** comando Importar configuração do estado atual da aplicação.

**Uso:**
```
$ drupal config:import [options] 
$ ci  
```

## Opções disponíveis
Opção | Detalhes
-------|-------------
--file | commands.config.import.option.file
--remove-files | commands.config.import.option.keep-files
