# module:install
**module:install** 命令 安装模块

**用法:**
```
$ drupal module:install [arguments] [options] 
```

## 可用选项
选项 | 详细
-------|-------------
--latest | Default to download most recent version

## 可用参数
参数 | 详细
---------|-------------
module | Module or modules to be enabled should be separated by a space
