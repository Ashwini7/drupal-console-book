# config:export:single
**config:export:single** 命令 导出单个配置

**用法:**
```
$ drupal config:export:single [arguments] [options] 
$ ces  
```

## 可用选项
选项 | 详细
-------|-------------
--directory | 导出配置存放目录
--include-dependencies | Export dependencies of the configuration as well.

## 可用参数
参数 | 详细
---------|-------------
config-name | 配置名称
