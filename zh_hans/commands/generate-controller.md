# generate:controller
**generate:controller** 命令 生成并注册一个控制器

**用法:**
```
$ drupal generate:controller [options] 
$ gcn  
```

## 可用选项
选项 | 详细
-------|-------------
--module | 模块名称
--class | 控制器类名
--title | Title of the controller
--method | The action method name
--route | 路由路径
--services | 从容器中导入服务
--test | 生成一个测试类
